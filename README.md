## High-precision currency tracking and instant conversion engine.

**Real-Time Exchange Rate (v2.2)** is a professional-grade Chrome Extension designed for seamless financial monitoring. Developed to provide real-time global currency data with a focus on speed, decimal precision, and non-intrusive UX.

[**Install from Chrome Web Store**](https://chromewebstore.google.com/detail/c%C3%A2mbio-em-tempo-real/ndkfobfkdpmlmkmnohgeibiaokgbioid)

### The Origin Story
I built this tool to solve my own problem: I couldn't find a currency tracker that was fast, minimalist, and actually kept the data saved without bloated features. What started as a personal optimization script quickly became a public tool after friends and colleagues started asking for it. Built for efficiency, scaled for the public.

### Core Features

#### Intelligent Monitoring
- **Smart Badge Persistence:** Real-time rate monitoring directly on the extension icon. Version 2.2 introduces state persistence, ensuring your preferred currency stays active even after browser restarts.
- **Auto-Refresh:** Market data updated every 10 minutes to ensure decision-making accuracy.

#### Advanced Conversion Engine
- **Selection-to-BRL:** High-performance context menu integration. Select any value on any website to convert it instantly to Brazilian Real (BRL) via a custom styled overlay.
- **Dual-Stream Input:** Simultaneous conversion between foreign currencies and BRL with support for 8-decimal precision and millisecond response times.
- **Data Export:** One-click copy function. Instantly copy the current rate, formatted with date and time, straight to your clipboard.
- **Market Insights:** Track daily High/Low peaks directly within the interface.

#### Supported Assets
Technical support for major global currencies including **USD, EUR, GBP, BTC, PLN (Złoty)**, and more.

### Technical Architecture & Public Staging

This repository acts as a **Public Staging and Documentation Mirror**. To protect the proprietary data-fetching algorithms and the "Marketing DNA" behind the price-tracking engine, the file structure is distributed as follows:

#### 🟢 Publicly Hosted (In this repository)
- **`manifest.json`**: Full extension architecture (Manifest V3).
- **`popup.html`**: The structural interface and UI layout.
- **`README.md`**: Technical documentation and project overview.

#### 🔴 Proprietary Logic (Excluded from Public Mirror)
- **`background.js`**: Service workers, badge update cycles, and context menu logic.
- **`popup.js`**: API integration, mathematical conversion engine, and data persistence.
- **`style` / `icons`**: Visual assets and core styling engine.

*The full source code is maintained in a private production environment for security and IP protection.*
